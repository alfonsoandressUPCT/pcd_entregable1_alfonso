# Gestión de Flota Espacial del Imperio Galáctico 🚀

Este repositorio contiene la primera práctica entregable de la asignatura Programación para Ciencia de Datos. El proyecto consiste en un software desarrollado en Python para gestionar el mantenimiento mecánico de una flota espacial.

## 🛠️ Tecnologías y Conceptos Aplicados

Las principales tecnologías y conceptos aplicados en este proyecto incluyen:

* **Lenguaje:** Python.
* **Paradigma:** Programación Orientada a Objetos (POO), explotando herencia, clases abstractas, enumeraciones y visibilidad.
* **Modelado:** Diagramas de clases, casos de uso y secuencia diseñados con mermaid.
* **Testing y Robustez:** Captura y tratamiento de excepciones, junto con pruebas unitarias usando `pytest`.
* **Control de Versiones:** Git con estrategia de ramas (`main` y `development`) y creación de tags.

## 📋 Descripción del Dominio

El sistema MiImperio permite a comandantes y operarios de almacén gestionar distintos elementos de la flota:

* **Naves:** Estaciones Espaciales, Naves Estelares y Cazas Estelares, cada una con características específicas como tripulación, ubicación o clase.
* **Unidades de combate:** Todas cuentan con un identificador de combate y una clave de transmisión cifrada.
* **Almacenes y Repuestos:** Control de stock, catálogos de piezas, proveedores y precios en distintas localizaciones.
